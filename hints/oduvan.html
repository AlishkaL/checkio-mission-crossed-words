Como iniciar a busca pela melhor letra de cruzamento entre as duas palavras?

Comece iterando sobre cada letra da palavra horizontal e, para cada letra, verifique se ela aparece na palavra vertical. Utilize o loop 'for' para tal ação.

Me parece que apenas um 'for' não será suficiente.

Você tem razão. Como devemos procurar uma letra que aparece nas duas palavras, vai ser necessário iterar pelas duas strings em busca de correspondências.

Preciso descobrir a posição da letra nas duas palavras. Como posso fazer isso?

Você vai precisar descobrir o índice. O método .index() da seguinte forma: palavra.index(letra).

Mas atenção! Caso a palavra tenha duas letras iguais, o método .index irá retornar o índice da primeira aparição da letra.

Como posso descobrir o índice independente do número de vezes que a letra aparece na palavra?

Para isso você deve usar uma variação do 'for' que usa o 'enumerate'. Seu funcionamento é igual, porém adicionamos outros parâmetros. Veja no exemplo abaixo: 

             
   for indice1, letra1 in enumerate(palavra, start=1):
       # indice1 vai retornar a posição da letra1 no momento da execução do 'for'
       # letra1 é a variável de iteração do 'for', funciona da maneira que já conhecemos
       # palavra é a palavra na qual vamos fazer as iterações, também funciona da maneira que já conhecemos
       # start é o número que começamos a contagem. Por padrão, start vale 0
       print(indice1, letra1) # use o print caso queira ver o funcionamento
